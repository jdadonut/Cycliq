# Cycliq
Cycliq is a multi-purpose open-source bot written in C#.

# Features
- Actions
- Moderation (soon)
- Disable specific command(s) in a guild/server (soon)
- Music (soon)
- Images
- Full integration with the nekos.life api! (in progress)
- And more! (soon)
